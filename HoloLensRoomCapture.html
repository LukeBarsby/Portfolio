This piece of work is for my PhD and is part of an initial study to create a Augmented Reality Capture tool for the design of interactive exhibits.

This application, named "ExhibitXR", is a tool to assit with Vitual Exhibitions. Specifically it can be used to design, create, and experience virtual exhibitions. This tool used Mixed Reality Tool Kit (MRTK), which is a set of libraries developed by Microsoft to assist in the HoloLens 2 creation workflow.

Here is a list of features the application has:

  1. Spatial Understanding of Environments
  2. The ability to allign digital Objects with real life objects
  3. Expandability of Assets: Textures, Models, and descriptions (more to come)
  4. Ability to create dynamic and interactable virtual exhibits
  5. Ability to save and view created exhibits locally to the HoloLens 2
