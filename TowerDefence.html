Tower Defence Game

This is a Tower Defence Game created in the Unity Engine, featuring different enemy types, turrets, upgrade paths and different levels.

This game was developed in my second year of university and is when I startd using the Unity Games Engines. I preferred this method of development greatly as it allowed me to focus on the game itself, rahter than building physics, rendering and loading systems to start development itself. This game included many systems and programming principels to aid in the structure and proficiency of the code. There was also a Fake Website built along side the game to aid in "promotion" . Systens and mechanics included:

  1. Inheritance and polymorphsm
  2. Multiple enemy tpyes with varied appearnces and abilities
  3. Object Pooling
  4. Game, sound, and level managers
  5. Multiple turrets with upgrade paths
  6. Placeable single use mines
  7. Particle effects and sounds played on game actions
