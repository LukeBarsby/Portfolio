Pacman

This was my first ever game created using S2D (Lightweight SDL2 Library), which included: Tile Maps, Simple AI, Abilities and win/loss conditions.

This was developed as I was learning C++ so no advance programming patterns or concepts were used. The most advanced concepts were classes and structs. Features of the game included:

  1. Win-lose conditions with relevant screens being displayed
  2. Simple AI
  3. Score tracking
  4. Tilemap display
  5. Abilities for Pacman when cherries were collected

Pacman Re-visited

WORK IN PROGRESS
