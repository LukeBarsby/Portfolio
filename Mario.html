Mario

This modules brief was a Mario platformer clone, this was programmed in C++ and used the SDL2 library. This was my most advanced project at the time which featured different enemy types, boss mechanics and multiple levels.

This was my second semester project in year 1 for Games Engine Creation. This included some advance principles including: inheritance, polymorphism and manual memory management through the use of pointers. The project itself featured an array of content:

  1. Multiple levels of exploration
  2. Multiple enemy types: Goombas, Koopa Troopas, and Bowser
  3. Full array of music interacting with game events
  4. A Special block (the pow box)
  5. Local 2-Player ability
