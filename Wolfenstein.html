Wolfenstein 3D

As a small challenge for myself, I decided to re-create Wolfenstein 3D as quickly as possible.

I do not own any rights to the sprites used for this project, It was created with no intent to share, publish or profit from.

For this project, I initially gave myself 48 hours, and during that time I got some good gameplay foundations in place. I had:

  1. All Textures and Materials created for the player, enemies and environments
  2. Re-creation of the level using Pro-Builder
  3. Simple door controllers
  4. Basic enemy Ai, that b-lined for the player on sight and had an idle when not in combat
  5. A Player Controller that could die, walk and shoot

Overall I was proud of what I achieved in the 48 hour time frame.

Wolfenstein Re-visited

Updates

I decided to re-visit its development of it. I accomplished a lot in the 48 hours but a lot was missing from the game and because of the timeframe I gave myself, a lot of the programming was in a "prototyping stage".

During the re-visit of the game I added all areas that were missing, in addition to, refactored the programming in the project. Items that were added were:

  • Sound Manager
  • Game Manager (Win/Lose & score/life keeping)
  • Pickup system
  • Inherited Weapon system
  • Inherited Enemy system With more Enemies
  • Secret rooms
  • Map generation

One part of this project that I liked was the map generation system I created. It took a while to block out the first level and during this time I wanted to create a streamlined process to generate maps in the future.

As this was a game re-creation, I had access to all the level designs in the game. For this map generation system, it takes in the level design images that can be sourced online, and then it iterates through the image comparing each cell to the correct block/object/Enemy and spawns it in at the correct position. The system could be improved but it successfully works in generating each map at playtime. This streamlines the level creation process with automatic placement of every wall, enemy, pickup, and door.
