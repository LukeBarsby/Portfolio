DrawFormer Prototype

This was a prototype for a mobile game I designed. However, this game never left its prototpying stage, this may be revisited in the future!

The core concept of the game is a casual puzzle game focusing on remembering the path to the exit.

Currently the prototype includes:

  1. Ability to run on Android devices and simulators
  2. Different difficulty modes, 3x3, 4x4, and 5x5 grids

Plans included for the Game:

  1. Focus on the theming of a Treasure Hunter
  2. Have different areas
  3. Add to the difficulty in later levels by adding traps: trip wires, spikes, single use floors, fire etc.
  4. Add ability for the player to pause their movement by holding a screen press to facilitate more gameplay addition
