My second Graphics based module introduced the 3D pipeline in Open GL. From this I learned how to create shapes from scratch, mark their positions, calculate normals and lighting, in addition to applying textures to them.

For this project I explored a variety of development environments and graphical principles, to create 2 and 3 dimensional objects. Other areas of the project were implementing algorithms, data structures and file handling using a high-level programming language and API to support games and graphics development. Tasks achieved in this project are:

  1. Loading and Rendering of Multiple Objects
  2. Animated/Moving 3D Objects
  3. Texturing
  4. Materials and Lighting
  5. Camera Movement
  6. User Input to control objects and/or camera
