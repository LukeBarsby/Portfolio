Final Year Project

For this project I explored Unreal Engine and created a first person shooter

As this was the first time I used Unreal Engine, a lot of time was spent familiarizing myself with the engine. During the project, I explored:

  1. In-game UI
  2. Implementation of engine input to control game elements
  3. Use of the Unreal Hierarchy to communicate between actors
  4. C++ development
  5. Blueprints Usage
  6. Combination of C++ and Blueprints Together
  7. AI behaviour trees to control game entities
  8. Particle systems
  9. Menu screens
  10. Win/Loss conditions
  11. FPS Gamemodes
